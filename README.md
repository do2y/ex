## 📦 자재 관리 시스템 (Material Management)
### 프로젝트 소개

> 자재 정보를 등록, 조회, 수정, 삭제할 수 있는 간단한 자재 관리 웹 서비스입니다.
> 사용자는 자재 목록을 확인하고, 상세 정보를 조회하거나 신규 자재를 등록할 수 있습니다.

<br>

### 주요 기능
**1. 자재 목록 조회**

- 전체 자재 리스트 확인

- 자재 코드, 이름, 분류, 단위, 재고, 상태 등 표시

- 최근 입고일 및 단가 확인 가능

**2. 자재 등록**

- 신규 자재 정보 입력

- 자재 코드, 이름, 분류, 규격, 단위, 단가 등 입력

- 현재고 / 안전재고 / 공급업체 / 창고 위치 관리

**3. 자재 상세 조회**

- 선택한 자재의 상세 정보 확인

- 재고, 단가, 공급업체, 담당자 정보 확인 가능

- 수정 및 삭제 기능 제공
